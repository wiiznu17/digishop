## Run service

At first run command:

```sh
npm install
```

### 1. digishop

In root path. Run the following command:

```sh
npm run dev:digishop
```

### 2. merchant

In root path. Run the following command:

```sh
npm run dev:merchant
```

### 3. portal

In root path. Run the following command:

```sh
npm run dev:portal
```
